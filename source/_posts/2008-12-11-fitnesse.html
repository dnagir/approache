Description

A tool for enhancing collaboration in software development by:

  • using wiki;
  • collaboratively defining AcceptanceTests in format of web pages stored on the server;
  • running tests and see results;
  • comparing customers' expectations to actual results;

 

Technicalities

  • Based on FitFramework.
  • Requires Java 5 to run.
  • Acts as a web server.
